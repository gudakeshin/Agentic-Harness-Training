# Architecting Guardrails: The 2026 Agentic AI Harness

This repository contains the interactive training portal for Finance Controllers, Record-to-Report (R2R) Teams, and Financial Systems Analysts to master the enterprise-grade mechanics of Agentic AI.

## Getting Started

1. **Install dependencies:**
   \`\`\`bash
   npm install
   \`\`\`

2. **Run the development server:**
   \`\`\`bash
   npm run dev
   \`\`\`

## Features Built-In
- Interactive Flowcharts
- API Sandboxing with JSON Policy testing
- Gemini API Integration (Auto-Generate Policies & Error Translation)

*Note on Gemini API:* To utilize the AI features locally, ensure an API key is provided to the \`callGemini\` function inside \`src/App.jsx\`.
